## 先介绍一下这个软件的功能

我做了一款专为网络安全人员提供服务的软件，有三大模块：

情报（主要是网络安全相关情报）分为：安全媒体、安全公众号、威胁情报、漏洞预警、数据泄露、军事情报、安全博客、人工智能等。程序会定期采集相关信息，保证情报及时准确！可对该情报进行评分、讨论、分享。

利器（主要是网络安全相关工具）分为：渗透常用、在线工具、搜索引擎、社工利器、入侵检测、密码破解、文件分析、威胁情报、在线靶机、网络攻击、移动安全、人工智能等。可对该利器进行评分、讨论、分享。利器分为内部和外部，内部是指APP内的工具，外部是指第三方提供的工具。

社区（主要是交流技术相关内容）分为：学习、问答、分享、求职、反馈、茶馆等。社区支持私聊和关注用户。点赞、点踩和评论帖子。

实验功能（内部APP工具开放使用，APP在我的-实验功能）：分为智能问答、智能绘画。就是大模型问答支持图片理解和对话，绘画就是AI生成图片。

## 重点：关于这个软件的延展性

这个软件我做了多端开发，分别是手机端（包括APP和小程序）、H5端、Web端、Windows桌面软件端。还做了后台管理系统。

该软件的技术栈简单说明（只列出大框架）：

- 前端：uniapp、vuejs（web前端框架vue-element-ui，后台系统vue-element-admin）、软件electronjs
- 后端：fastapi（+celery+redis）

所有端代码风格统一，代码量低，维护量低，开发效率高。这套系统可以适配多种不同的业务场景，非常方便个人开发者极速开发，这也是我一开始做这个软件的初心，为了做一套多端统一系统。

目前基础功能已完成，待后续慢慢完善。有部分分类还未添加数据暂时隐藏了起来。
